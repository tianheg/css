How CSS Work?

  1. 浏览器加载 HTML
  2. 将 HTML 转化为 DOM
  3. 将 HTML 中链接的 CSS 从服务器下载下来
  4. 解析获得 CSS 文件，根据不同的选择器进行分类，对各选择器进行渲染，形成渲染树
  5. 展示在屏幕上

DOM 是一种树状结构。HTML 中的所有元素、属性和文本等都会在 DOM 中成为一个个节点，由于元素、属性和文本等是存在级别的，所以由此形成了一个 DOM 节点树。
